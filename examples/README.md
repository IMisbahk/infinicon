# Examples

## Quick start (zero config)

From repo root:

```bash
bun install
bun start
```

Uses [`simple-chat.ts`](simple-chat.ts) — one file, root `.env`, no setup.

## Other examples

| Command | Description |
|---------|-------------|
| `bun start` | Server + simple chat (recommended) |
| `bun run example:simple` | Chat only |
| `bun run example:agent` | Full recall + consolidation demo |
