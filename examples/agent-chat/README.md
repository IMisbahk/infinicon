# Agent chat example

Full SDK demo with recall + consolidation. For the minimal path use `bun start` at repo root.

```bash
# from repo root — .env already exists after bun install
bun run dev              # terminal 1 (or use bun start for all-in-one)
bun run example:agent    # terminal 2
```

Same env as everything else: root [`.env`](../../.env.example).
