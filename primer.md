## What I know so far
Infinicon is a spec-first memory runtime for AI agents focused on durable memory semantics, provenance, and bounded context assembly. The repository currently contains architecture docs, ADRs, and v0 draft specs, with implementation intentionally deferred until specs stabilize. Core direction is storage/model/framework agnosticism with explicit lifecycle, consistency, and tombstone behavior.
